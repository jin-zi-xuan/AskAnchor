# AskAnchor

AskAnchor 是一个 Chrome Extension Manifest V3 扩展，用来增强主流 AI 网页中的阅读、追问和原文回溯体验。

当前 MVP 不接 API、不打开新窗口、不弹解释面板。用户在 AI 回答里选中一段文字后，AskAnchor 会在选区旁生成“追问这一段”按钮；点击后可选择追问模板，再把对应 prompt 填入当前 AI 的输入框，同时把选中的文字保存为可回跳锚点。

## 支持站点

- ChatGPT: `chatgpt.com`, `chat.openai.com`
- Gemini: `gemini.google.com`
- Claude: `claude.ai`
- Perplexity: `perplexity.ai`
- Poe: `poe.com`
- Microsoft Copilot: `copilot.microsoft.com`
- DeepSeek: `chat.deepseek.com`
- Kimi: `kimi.moonshot.cn`
- 豆包: `doubao.com`
- 通义: `tongyi.aliyun.com`
- 文心一言: `yiyan.baidu.com`

## 隐私与权限

AskAnchor 默认在上面的支持站点运行，因为它需要读取页面中的 AI 消息、在页面内插入追问按钮和锚点、寻找当前页面输入框，并在必要时复制追问内容。

- AskAnchor 不上传聊天记录。
- AskAnchor 没有后端，也不会把聊天内容发送到远程服务器。
- 锚点只保存在当前页面的 `sessionStorage`，刷新后会尽量恢复，关闭会话后由浏览器清理。
- 小猫位置只保存在当前页面的 `localStorage`。
- 平台开关和页面行为设置只保存在浏览器本地的扩展存储中。
- 插件默认在支持的 AI 网站页面运行；你可以在扩展设置页关闭任意平台。
- `clipboardWrite` 只用于找不到输入框时，把你主动生成的追问内容复制到剪贴板。

## 当前功能

- 在 AI assistant 回答中选中文字。
- 只在有效选区附近显示“追问这一段”按钮。
- 点击后选择追问模板，生成包含选中文本的轻量追问。
- 自动把追问 prompt 填入当前 AI 页面的输入框。
- 同时创建锚点，锚点名称默认使用用户选中文字，过长时自动截断。
- 用户发送追问后，AskAnchor 自动跳转到刚发送的那条追问问题。
- 打开支持的 AI 页面后，3D 小黑猫会常驻趴在聊天框上，点击可展开当前对话解释过的内容，也可以沿聊天框上沿拖动到顺手的位置。
- 时间轴是独立的侧边对话导航，用于快速定位当前聊天里的每一轮问题。
- 锚点列表按当前对话隔离，不会混入其他聊天窗口的锚点。
- 小黑猫偶尔会眨眼和动耳朵，点击小黑猫可打开锚点列表。
- 锚点列表可以点击，点击后准确回到原来选中的文字位置。
- 锚点列表支持删除、列表内改名，以及“未解决 / 已理解”状态切换。
- 锚点定位会保存“选中文字 + 前后文 + 文本偏移”，刷新后也会尽量恢复到原文位置。
- 如果暂时找不到输入框，会把 prompt 复制到剪贴板作为兜底。

## 交互设计

AskAnchor 借鉴了 `chatgpt-gemini-timeline` 的几个成熟思路，但没有复制其 GPL 代码：

- 选区追问：选中文字后出现轻量操作按钮。
- 文本定位：用 exact/prefix/suffix 这类文本锚点信息恢复原文位置。
- 时间轴导航：用侧边短刻度承载当前聊天的对话轮次，减少对聊天窗口的遮挡。
- 输入框跟随：3D 小黑猫默认蹲在当前 AI 输入框右上角，拖动后会记住它在聊天框上的相对位置，隐藏时缩到侧边只露出一部分。

## 本地安装

1. 打开 `chrome://extensions`。
2. 开启 Developer mode。
3. 点击 Load unpacked。
4. 选择本项目文件夹。
5. 打开任一支持站点，在 AI 回答中选中文字，点击“追问这一段”。

修改代码后，需要在 `chrome://extensions` 里点击 AskAnchor 的 reload，然后刷新 AI 网页。
完整设置页可以从扩展详情页打开，或在 popup 中点击“打开完整设置”。默认所有平台均开启，可在设置页手动关闭。

## 使用方式

1. 在 AI 回答中选中你想追问的文字。
2. 点击选区旁边的“追问这一段”。
3. 在小菜单里选择“解释概念”“举个例子”“指出代码问题”等追问模板。
4. AskAnchor 会把对应追问内容填入当前 AI 的输入框。
5. 你检查一下内容，然后点击当前 AI 页面的发送按钮。
6. 发送后页面会自动跳到刚发出的追问问题。
7. 需要回到原文时，点击小黑猫并选择对应锚点。
8. 锚点看完后，可以在列表中标记为“已理解”；不需要保留的锚点可以直接删除。
9. 需要快速定位某轮对话时，点击右侧时间轴刻度。
10. 双击小黑猫可以把它隐藏到侧边，继续浏览页面。

## Prompt 格式

```text
【选中内容】
...

请解释我在上文中选中的这段内容，并说明它和上下文的关系。
```

也可以选择举例、换一种说法、代码问题、翻译并解释、反驳检查或自定义提问等模板；“自定义提问”会在输入框里保留空白的“我的问题：”，方便直接补写。

## 项目结构

- `manifest.json`: Chrome MV3 配置、权限和内容脚本匹配规则。
- `contentScript.js`: 文本选择监听、原地追问、锚点记录、发送后定位、时间轴和小猫入口。
- `styles.css`: 选区按钮、小猫入口、锚点列表、时间轴、顶部提示和高亮样式。
- `assets/askanchor-cat-perch.png`: 小猫展开状态图片资源。
- `assets/askanchor-cat-peek-right.png`: 小猫收起状态图片资源。
